# Selenium Practice Dashboard

A Next.js 16 app with interactive UI modules for Selenium practice scenarios:

- Alerts
- Calendar
- Checkbox
- Dropdown
- File Upload
- Forms
- Frames
- Mouse Events
- Radio Button
- Suggestion List
- Waits
- Windows

## Setup

```bash
npm install
```

## Run

```bash
npm run dev
```

Visit `http://localhost:3000`.

## Scripts

- `npm run dev` - start development server
- `npm run build` - production build
- `npm run start` - run production server
- `npm run lint` - run ESLint

## Tech Stack

- Next.js 16 (App Router)
- React 19
- Tailwind CSS 4
- TypeScript
